## 4. 计算公式

### 4.1 SMA (简单移动平均)

```
SMA(n) = (P₁ + P₂ + ... + Pn) / n
```
其中 P = 收盘价，n = 周期

### 4.2 EMA (指数移动平均)

```
EMA(t) = P(t) × α + EMA(t-1) × (1-α)
α = 2 / (n + 1)
```

### 4.3 MACD

```
EMA12 = 12日指数移动平均
EMA26 = 26日指数移动平均
DIF = EMA12 - EMA26
DEA = DIF的9日EMA
MACD柱 = 2 × (DIF - DEA)
```
- DIF > DEA → 多头信号 (红柱)
- DIF < DEA → 空头信号 (绿柱)

### 4.4 布林带 (BOLL)

```
中轨 = SMA(20)
上轨 = 中轨 + 2 × 标准差(20)
下轨 = 中轨 - 2 × 标准差(20)
```
- 价格触及上轨 → 超买可能
- 价格触及下轨 → 超卖可能
- 带宽收窄 → 变盘信号

### 4.5 RSI

```
RS = n日内平均上涨幅度 / n日内平均下跌幅度
RSI = 100 - 100 / (1 + RS)
```
- RSI > 70 → 超买
- RSI < 30 → 超卖
- RSI 背离 → 趋势反转信号

### 4.6 KDJ

```
RSV = (收盘价 - n日内最低) / (n日内最高 - n日内最低) × 100
K值 = 2/3 × 前K值 + 1/3 × 当前RSV
D值 = 2/3 × 前D值 + 1/3 × 当前K值
J值 = 3 × K值 - 2 × D值
```
- K > D → 看涨
- K < D → 看跌
- K/D > 80 → 超买
- K/D < 20 → 超卖

### 4.7 威廉指标 (W%R)

```
W%R = (n日内最高价 - 收盘价) / (n日内最高价 - n日内最低价) × (-100)
```
- W%R > -20 → 超买
- W%R < -80 → 超卖

### 4.8 乖离率 (BIAS)

```
BIAS(n) = (收盘价 - SMA(n)) / SMA(n) × 100%
```
- BIAS(5) > 5% → 超买
- BIAS(5) < -5% → 超卖

### 4.9 ATR

```
TR = max(当日最高-当日最低, |当日最高-前日收盘|, |当日最低-前日收盘|)
ATR(n) = TR的n日移动平均
```
- ATR越大 → 波动越大
- 用于止损设置和仓位管理

### 4.10 MFI (资金流向指标)

```
典型价格 = (最高 + 最低 + 收盘) / 3
资金流 = 典型价格 × 成交量
若今日典型价格 > 昨日 → 正资金流, 否则负资金流
MFR = n日正资金流之和 / n日负资金流之和
MFI = 100 - 100 / (1 + MFR)
```
- MFI > 80 → 超买
- MFI < 20 → 超卖
