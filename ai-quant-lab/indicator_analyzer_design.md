# Stock Indicator Interactive Analyzer (SIIA) 设计文档 v1.0

## 1. 概述

**项目**: task02_indicator_lab — 交互式技术指标分析工具
**文件**: `indicator_analyzer.html`
**描述**: 一个纯前端的交互式HTML工具，支持选择股票数据、调节指标参数、实时重绘图表

### 核心功能
1. **股票切换** — 从预置股票列表中切换，图表自动刷新
2. **4大指标计算** — RSI、MACD、布林带、ATR
3. **参数可调** — 每个指标支持滑块/输入框调节参数，实时重绘
4. **交互式图表** — 使用ECharts渲染，支持缩放、悬停提示
5. **数据内嵌** — 预置三只A股日线数据，无需后端

## 4. 指标计算公式

### 4.1 RSI
- 参数: period (默认14, 范围2-50)
- 公式: `RSI = 100 - 100 / (1 + avgGain/avgLoss)`
- 辅助线: 超买线70, 超卖线30

### 4.2 MACD
- 参数: fastPeriod (默认12), slowPeriod (默认26), signalPeriod (默认9)
- 公式:
  - EMA_fast = EMA(close, fastPeriod)
  - EMA_slow = EMA(close, slowPeriod)
  - DIF = EMA_fast - EMA_slow
  - DEA = EMA(DIF, signalPeriod)
  - MACD = 2 × (DIF - DEA)

### 4.3 布林带
- 参数: period (默认20), multiplier (默认2.0, 范围1.0-4.0)
- 公式:
  - MID = SMA(close, period)
  - STD = STD(close, period)
  - UPPER = MID + multiplier × STD
  - LOWER = MID - multiplier × STD

### 4.4 ATR
- 参数: period (默认14, 范围2-50)
- 公式:
  - TR = max(H-L, |H-PrevC|, |L-PrevC|)
  - ATR = SMA(TR, period)

## 6. 交互逻辑

### 6.1 参数变化 → 重绘
```
用户调节滑块 → 参数更新 → 清除旧图表 → 重新计算指标 → ECharts setOption()
```

### 6.2 图表切换
```
用户选择指标checkbox → 显示/隐藏对应指标曲线 → setOption() 更新
```

### 6.3 时间范围切换
```
点击时间按钮 → 截取数据子集 → 清除旧图表 → 重算全部选中指标 → 重绘
```

### 6.4 ECharts 特性
- 悬停十字线 (axisPointer)
- 图例可点击切换
- 区域缩放 (dataZoom)
- 自适应窗口resize

## 9. 设计原则

1. **单文件** — 一个 `indicator_analyzer.html` 包含所有CSS/JS/数据
2. **零后端** — 完全在浏览器中运行
3. **手写计算** — 所有指标在JS中手写实现，不依赖第三方库
4. **响应式** — 适配宽屏/笔记本/平板
5. **即时反馈** — 滑块拖动时实时重绘，无需点击按钮
